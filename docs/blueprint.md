## Core Features:

- User Authentication: Student login via Google OAuth or Student Email.
- Profile Management: Create/update profile: Name, Year, Department, Vehicle details, Carpool preference.
- Commute Options Display: View all commute options with filters (fastest, cheapest, eco-friendly).
- Carpool Request System: Request to join available carpools; receive notifications for approvals or updates.
- Route Planner: Route planner with Google Maps/OpenStreetMap showing multiple options, estimated time, distance, and cost.
- City Guide: City guide for non-local students with popular destinations and travel suggestions.
- Smart Recommendations: Smart recommendations highlighting available carpools and preferred routes.
- Admin User Management: Manage student profiles: view, edit, deactivate accounts.
- Vehicle & Carpool Monitoring: Monitor vehicles and carpool participation.
- Schedule Management: Manage College Bus and RCT bus schedules and fare info.
- Destination Management: Add/edit/delete city destinations and route info.
- Notification System: Send broadcast notifications or alerts to students.
- Analytics & Reporting: Analytics & reporting: rides per student, popular destinations, commute patterns.

## Style Guidelines:

- Interactive & responsive UI for both user and admin dashboards.
